# SpartaMarket-DRF
## 🖥️ 프로젝트 소개
### 스파르타 마켓 : 우리를 위한 중고거래가 이루어지는 커뮤니티 작성 DRF(Django RESTful Framework)를 이용한 중고거래 플랫폼을 구현하여 상품 관련 페이지와 회원 관련 페이지를 만들고자 함.

### 🕰️ 개발 기간
2024.04.26. ~ 2024.05.02.

### ⚙️ 개발 환경
Python : Django, DRF


DB : SQlite
